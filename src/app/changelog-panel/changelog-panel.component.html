Change Log

v1.0.42 Gaining large numbers of items at once is now much more efficient, so having obscenely huge farms no longer breaks the game.
v1.0.41 Panel size fixes. Bloodline ascension rank 5. Fixed log line duplication bug. Pause on death should no longer allow extra days to pass before pausing.
v1.0.40 Changed the way damage and armor work. Late-game fights should now be more epic and interesting.
v1.0.39 Better descriptions for resting activities. Shift-click activities to load them onto the schedule 10 at a time. Canopy beds give more health.
v1.0.38 Better performance for aspiring immortals who want to buy vast tracts of land.
v1.0.37 New status bars that show values on non-chrome browsers.
v1.0.36 You can now choose how much money to reserve when doing automatic purchases.
v1.0.35 The log panel can now be resized by sliding the slider up and down. The Ascension Techniques panel stays unlocked once unlocked. Old junky gems can now be easily removed if you know how.
v1.0.34 Style fix, clear schedule button, and auto-equip can now be turned off in the options.
v1.0.33 Some styling improvements and a fix to very late game reward loot.
v1.0.32 Automerging will now upgrade equipped weapons and armor if all merge manuals have been achieved and purchased.
v1.0.31 Some balance changes.
v1.0.30 A certain impossible task is now slightly less impossible.
v1.0.29 Silly levels that stamina could reach are now slightly less silly. Pausing the game banks time ticks.
v1.0.28 Improvements to automatic home purchasing.
v1.0.27 You can now save and load an activity schedule.
v1.0.26 Fixed a bug with activities incorrectly being removed from the schedule on reincarnation.
v1.0.25 New activities and activity upgrades now get log entries. Armor now degrades when attacked.
v1.0.24 Bug fix for aspiring immortals who got tangled up while trying to swim to the bottom of the ocean.
v1.0.23 Bug fix for aspiring immortals who achieved infinite wealth. Money is now capped at an amount that is plenty for everything but doesn't break your computer.
v1.0.22 Leatherworking and Woodworking now have master ranks. You can now see how much money you made on each job the last time you did it. Better logs can now be chopped.
v1.0.21 Followers can now be trained to be more powerful and can be automatically dismissed by job.
v1.0.20 Added Discord link.
v1.0.19 Added changelog.
v1.0.18 New follower types are now available.
v1.0.17 Crafting log entries can now be filtered out.
v1.0.16 Added tutorial help button.
v0.0.0 - v1.0.15 Created and launched the game.
